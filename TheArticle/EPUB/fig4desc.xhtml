Figure 4, Growth rate of tail relative to body data point plot.

SVG Description

Graph showing the plot of tail length relative to body length on a logarithmic scale. The graph has numerous points plotted and a straight linear regression line that demonstrates the points on the plate have a nearly linear relationship.

The vertical axis has a logarithmic scale of the tail length in millimeters, ranging from about 1.25 to about 2.15. The horizontal axis has a logarithmic scale of the body length in millimeters, ranging from about 1.40 to about 2.05.

The point are green circles and the linear regression line is purple.

The graph should not be considered a source of precise accurately measured data points, the graph was traced from a scanned images of a printed graph that itself did not appear to have precision implied. It provides a good feel for the growth rates but this graph should not be considered a source data set.
